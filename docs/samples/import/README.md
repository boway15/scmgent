# 数据导入示例 CSV

用于 **数据中心 → 数据导入**（`/data/import`）的演示数据。商品均为 **跨境家具品类**（客厅、书房、卧室、灯饰等），SKU 编号与本地库中演示数据一致。

## 导入顺序

按文件名序号依次上传（SKU 必须先导入）：

| 文件 | 导入类型 | 验证页面 |
|------|----------|----------|
| `01-skus.csv` | SKU 主数据 | 库存总览 |
| `02-inventory.csv` | 库存盘点 | 库存总览 |
| `03-sales.csv` | 销量历史 | 补货建议 / 补货预测 |
| `04-safety_stock.csv` | 安全库存（含 `warehouse_code`） | 安全库存设置、缺货预警 |
| `05-pmc_plans.csv` | PMC 工厂计划 | PMC 计划列表 |
| `06-compliance.csv` | SKU 合规属性 | 合规总览 / SKU 合规 |

## PMC 计划额外字段

上传 `05-pmc_plans.csv` 时，在页面上填写：

- 计划名称：`2026年6月家居工厂排产`
- 计划日期：`2026-06-01`
- 交期：`2026-07-15`

## 使用方式

1. 打开 http://localhost:8080/data/import
2. 选择对应导入类型
3. 点击「上传文件」选择 CSV，或复制文件内容粘贴到文本框
4. 导入完成后到上表「验证页面」查看效果
