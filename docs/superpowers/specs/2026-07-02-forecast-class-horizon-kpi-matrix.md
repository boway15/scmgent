## 6. 使用说明

1. **当前（v3）**：只有 A·主力 1–3 月实测 **42.6%**，距建议 KPI 15% 差 **27.6pp**——**不可将 15%/25% 作为当前发布门禁**。  
2. **Phase 1 完成前**：建议 KPI 仅作参考，门禁只用「D 类零误预测 = 0」。  
3. **Phase 3 完成后**：A·主力 1–3 月 **≤15%**、3–6 月 **≤25%** 可作为硬门禁。  
4. **管理看板**：按本矩阵展示「当前值 / 建议 KPI / 达标状态」，避免全库单一 MAPE。
