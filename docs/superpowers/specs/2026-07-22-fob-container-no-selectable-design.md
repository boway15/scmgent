# FOB 平账结果 · 货柜号可复制 · 设计说明

**日期**：2026-07-22  
**状态**：待实现  
**关联**：`apps/web/src/components/FobContainerMatrixPanel.tsx`、`.cursor/rules/fob-settlement-dev.mdc`  
**触发**：平账矩阵「按柜平账与调账」列表中货柜号（如 `BMOU5120438`）无法选中复制

---

## 1. 背景与问题

平账矩阵父行将「箭头 + 平账状态图标 + 柜号」包在同一个 `<button>` 内，用于展开/收起子行。浏览器对 button 内文本的选中与复制支持差，业务人员无法把柜号粘贴到检索、对账或沟通工具。

---

## 2. 决策摘要

| 项 | 决策 |
|----|------|
| 方案 | 柜号移出展开按钮，支持鼠标拖选后系统复制（Ctrl+C / 右键） |
| 展开/收起 | 仍由左侧箭头（及状态图标区）触发 |
| 一键点击复制 | 不做 |
| 复制图标 / toast | 不做 |
| 副标题（工厂数 / 业务编号） | 不改 |
| 改动文件 | 仅 `FobContainerMatrixPanel.tsx` |

---

## 3. 交互设计

### 3.1 父行柜号单元格

```
[ ▶/▼ ] [ ✓/! ]  BMOU5120438
                  2 工厂/主体 · 业务编号 …
```

- **箭头（可含状态图标）**：点击切换展开/收起；保持现有默认展开规则（未平账默认展开等）。
- **柜号文本**：普通可选中文本（`font-mono`）；用户拖选后用系统复制即可。
- **悬停提示**：柜号悬停仍展示「柜号 / 工厂/主体 / 业务编号 / 工厂数」多行 tip，行为不变。

### 3.2 明确不做

- 点击柜号写入剪贴板
- 柜号旁 Copy 图标
- 全局 toast / 通知组件
- 其他页（分摊基数表、柜号匹配诊断列表等）的柜号复制能力（本次不扩展）

---

## 4. 实现要点

1. 拆开当前包住整块内容的单个 `button`：
   - 可点击区域：仅箭头，或「箭头 + 状态图标」；`type="button"`，调用现有 `toggleExpanded`。
   - 柜号：放在 button 外，继续使用现有 `TruncatedTip` 包裹展示。
2. 布局与样式尽量对齐现状（间距、`pl` 与副标题缩进、sticky 列宽不变）。
3. 不新增依赖、不改 API / 数据结构。

---

## 5. 验收标准

- [ ] 在「按柜平账与调账」列表中，可鼠标拖选货柜号并复制到剪贴板。
- [ ] 点击左侧箭头仍可展开/收起该柜明细行。
- [ ] 平账状态图标展示与悬停 tip 行为与改前一致。
- [ ] 未引入点击复制、Copy 图标或 toast。

---

## 6. 测试建议

手工验收即可（纯 UI 结构拆分，无业务逻辑变更）：

1. 打开任一有分摊结果的 FOB 批次 → 分摊核算 / 平账矩阵。
2. 拖选某柜号 → 复制 → 粘贴到记事本，内容与界面一致。
3. 点击该行箭头，确认展开/收起仍正常。
